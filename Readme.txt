项目名称：哈福网商城
需要链接互联网iconfont使用的是阿里巴巴的在线CSS

首页地址:index.html
此外还有注册、登录、商品列表页、商品详情页、购物车。
全项目采用ES6模块化编程，采用图片整合，iconfont字体。使用JQuery框架，Cookie插件，validtae插件等。
（详情页的'立即购买'和'三级导航',其余按钮基本都添加了对应功能）

主页：
	    三级导航：纯CSS样式。
	    仿淘宝的楼层floor
	    轮播图：简单轮播图
	    获取JSON数据展示
	    搜索框功能只能记录搜索记录（搜索按钮跳转列表页没有实现）

商品列表页面：
	   入口主页轮播图上方的导航栏
	   商品列表页面采用数据按需加载的模式，客户需要浏览更多商品时候可以点击“加载更多”。

商品详情页：
	    放大镜功能
	    加入购物车按钮

购物车页面：
	    需要登陆才能浏览
	    购物车上的价格数量，逻辑运算全部编写完成
	    点击商品名称，会跳转对应商品详情页。
    
注册页面：
     	调用接口，简单的注册，注册成功5秒跳转登录页。

登陆页面：
     	调用接口，简单的注册，注册成功5秒跳转登录页。
